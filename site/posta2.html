Trabajo 02. Javascript

Minijuego en Phaser

Agosto 31, 2020 publicado por Daniel Baez

Aqui podemos observar el resultado de la actividad numero 2, que consiste en hacer uso de Phaser para hacer un minijuego.

Phaser es una herramienta de programación, más concretamente, una librería Javascript, creada por Richard Davey, destinada a desarrollar juegos de una forma rápida para navegadores web, válida tanto para versiones de escritorio como móvil, y pueden ser compilados para usarlos y publicarlos como aplicación móvil. El juego se carga en un Canvas o en WebGL.
Canvas es un elemento HTML que es utilizado para dibujar gráficos, hacer composiciones de fotos o incluso realizar animaciones.
WebGL es una evolución de Canvas que permite cargar gráficos 3D.

Se tomo la base del tutorial y se personalizo con una tematica libre, en este caso opte por hacerlo de Los Simpsons con un Homero tratando de comer todas sus "preciadas" rosquillas. El juego termina cuando Homero toca una cerveza.
Mi puntaje maximo han sido 270 puntos.

*Se desea realizar unas modificaciones más adelante para hacer que las plataformas tambien sean dinamicas como las cervezas.
